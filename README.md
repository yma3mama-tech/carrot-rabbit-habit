# carrot-rabbit-habit

目標を達成してうさぎに人参をあげるアプリ

## 仕様

* ブラウザの保存機能を使っているので、データは使った人のスマホやPCの中だけに保存されます。

## 使い方

* 目標を入力します。
* 達成できたら人参ゲット。うさぎにあげると喜びます。
* 達成できた日にはカレンダーに人参マークがつきます。
* ストーリーを残すことができ、消去もできます。
* うさぎをなでるとメッセージが表示されます。
